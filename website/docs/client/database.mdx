rex go client supports `database/sql` methods, including

- `Exec()`
- `ExecContext(ctx)`
- `Query()`
- `QueryContext(ctx)`
- `QueryRow()`
- `QueryRowContext(ctx)`
- `Prepare()`
- `PrepareContext(ctx)`
- `Tx(ctx)`

```go
client.Query("select * from users")
client.Exec("delete from users")
```
